## Conventions

| Dossier | Signification |
| :--- | :--- |
| `Boucle_X_*` | Une boucle d'apprentissage (Prosit) avec son CER (compte-rendu LaTeX → `main.pdf`). |
| `WorkShop/` | Ateliers pratiques et code produit pendant la boucle. |
| `_Project_*` | Le projet fil rouge du module. |
| `_Report/` | Rapports et livrables transverses au module ou à l'année. |
| `_Template/` | Gabarit LaTeX commun (préambule, styles, assets). |
