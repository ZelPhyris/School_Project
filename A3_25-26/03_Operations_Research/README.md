## Projet — [Green VRP Algorithms](./_Project_Green-VRP-Algorithms/)

Résolution d'une variante du **Vehicle Routing Problem (VRP)** dans le cadre d'un appel d'offres ADEME (CesiCDP) : calculer une tournée optimale entre un sous-ensemble de villes avec retour au point de départ, en minimisant la durée totale (et donc la consommation et les émissions).

- Langage : Python 3 (PEP 8). Format : Jupyter Notebooks.
- Méthodes : modélisation mathématique, analyse de complexité, métaheuristiques (ex. ACO).

## Compétences clés

- Modélisation mathématique de problèmes d'optimisation.
- Analyse de complexité temps/espace.
- Conception d'heuristiques et de métaheuristiques.
- Étude statistique des performances expérimentales.
