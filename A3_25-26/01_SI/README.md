# Systèmes & Infrastructure (SI)

Ce module couvre l'administration et la sécurisation d'un système d'information complet : du déploiement d'une infrastructure virtualisée jusqu'à sa supervision, en passant par l'annuaire, la sauvegarde, le filtrage et le chiffrement.

## Projet — [`_Project/`](./_Project/)

Projet d'infrastructure synthétisant les compétences du module (rapport disponible dans [`Report/`](./_Project/Report/)).

## Compétences clés

- Conception et déploiement d'une infrastructure virtualisée.
- Mise en place d'un annuaire Active Directory (ADDS).
- Sécurisation réseau : filtrage, chiffrement, défense en profondeur.
- Sauvegarde, supervision et continuité de service.
