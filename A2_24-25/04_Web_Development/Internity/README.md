# Internity Internship Search Platform

## Objective
To facilitate the search for internships for students by centralizing offers and information from companies recruiting interns.

## Main features
- Search for offers: Access to an up-to-date database of internship offers.
- Company directory: List of companies that have already recruited interns or are actively searching.
- Application management: Ability to save and track applications.

## Technologies used
HTML, CSS, JavaScript, PHP, MySQL

## Why this project?
Students often have to search for internships by activating their personal and professional networks (LinkedIn, previous promotions, etc.). This platform aims to simplify and structure this process by bringing all the information together in one place.
